# Code of Conduct

## Our Pledge

We want this project to be a respectful and practical place to collaborate.

## Expected Behavior

- Be direct and constructive.
- Focus feedback on the work.
- Respect different levels of experience.
- Avoid harassment, insults, and personal attacks.

## Unacceptable Behavior

- Discriminatory language or behavior
- Harassment or intimidation
- Publishing private information without permission
- Bad-faith disruption of project discussions

## Enforcement

Project maintainers may remove comments, close discussions, or block contributors who violate this code of conduct.
